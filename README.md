# MIA - AI Therapist & Friend Companion

An AI companion that acts as both a friend and therapist, trained to provide empathetic, natural conversations.

##  Project Goal

Build a conversational AI that feels like chatting with a supportive friend and companion with focus on humor and wit who also has therapeutic capabilities.

## Current Status

- **Version:** 2.0 (GPT-2 fine-tuning)
- **Training Data:** 150 conversation pairs
- **Model:** GPT-2 (fine-tuned)
- **Framework:** TensorFlow, Hugging Face Transformers
- **Status:** In Development

##  Project Evolution

### Version 1.0 - LSTM Model
- Basic LSTM neural network
- 30 training examples
- Output: Generic responses
- **Outcome:** Functional but not natural

### Version 1.1 - GPT-2 Fine-tuning
- Pre-trained GPT-2 model
- 150 training examples
- Natural language generation
- Friend-like personality
- **Status:** Current development
